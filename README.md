**CONOCER MIS CREDENCIALES EN GIT HUB**

Para poder empezar con todo el proceso debimos comenzar por saber cual es nuestro nombre de usuario y como aparece nuestra cuenta en git hub, también tener muy clara cual es mi contraseña.

**CONOCER MI URL DE GITHUB**

Debemos estar enterados totalmete de cual es nuestro URL para ser buscados en github. **Por ejemplo:** Mi url es: **https://github.com/Katherin-av**

**¿CÓMO CREAR UN REPOSITORIO DE GITHUB?**
1. Nos dirijimos a el espacio **Repositorios**.

2. Ya estando en repositorios se oprime el botón **New**.

3. Luego se despliega una especie de formulario donde nos pide que le pongamos un nombre al repositorio que estamos creando y también una breve descripción del repositorio (Este espacio no es obligatorio).

4. Después de tener esa información le confirmamos crear el repositorio.

5. Nos aparece la opción de crear oficialmente el repositorio y nos lleva directamente a un Visual Studio Code que está integrado en github.
